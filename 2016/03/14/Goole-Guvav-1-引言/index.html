笔记

Goole Guvav 1 引言

　　之前在一个新项目中用到过guava，当时是参考大牛的代码匆匆用，没有细研究，事后觉得很强大，可以把java代码写得的很简洁漂亮，因为公司旧框架还是用远古的 java 1.5，后来的开发就没有接触过。偶然发现大牛离职前留给我的资料中有份 Getting Started with Google Guava ，于是就自己硬着头皮看了英文版，发现也能看懂大概意思，开篇有句 :To start working with Guava, all you need to have is Java 1.6 or a higher version
installed. ,觉得自己专业知识真得该更新。感慨外面的技术日新月异，一日千里。不能再像青蛙那样坐进观天，最后温水煮挂。
　　所以打算抽些时间在这把guava 的学习笔记整理下,深入学习的同时以便日后有用到时可以查阅。 　

引言

　　Guava 工程包括了若干被Google的java项目广泛依赖的核心库，例如：集合[collections]、缓存[caching]、原生类支持[primitives support]、并发库[concurrency libraries]、通用注解[common annotations]、字符串处理[string processing]、I/O等等。所有这些工具每天都在被Google的工程师应用在产品服务中。这些很前卫技术应该引用到日常工作里，要与时俱进。

　　因为Google被墙掉了， 无法连接到Guava的API文档，有些不清晰的地方，总是很头疼。
